Device events

This page shows the events, sent by a BBC micro:bit Bluetooth Mesh node to Drogue IoT, and received from Drogue IoT.

The device will periodically publish sensor data, battery level as well as the current button states.

When you press the buttons on the micro:bit, it will emit a button status event, which you can see on the left side.

You can also control the LED display on the micro:bit by enabling/disabling blinking of the display, as well as adjusting the brightness level.

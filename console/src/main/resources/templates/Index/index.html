Device events

This page shows the events, sent by your BBC micro:bit Bluetooth Mesh node connected to Drogue IoT. The device will periodically publish sensor data, battery level as well as the current button states.

When sensor data arrives, or you press the buttons on the micro:bit, it will emit an event, which you can see on the left side.

You can also control the LED display on the micro:bit by enabling/disabling blinking of the display, as well as adjusting the brightness level.

Claim device

This page allows you to claim a device by entering the ID on a sticker on the micro:bit box. Make sure your device is powered on when attempting to claim it.

Once claimed, you should be able to see events from the micro:bit propagating to this console, as long as it is within the range of one of the gateways.
